# histoire_manettes


Si vous voulez mon avis, le desing des manettes utilisées pour les consoles de jeux ont une histoire aussi intéressante que les consoles en elles-mêmes. Je pense que la meilleure manière de prouver mon point serait de commencer avec le commencement. Cette présentation va parler de l'évolution des manettes de consoles de jeux vidéos, en passant par celles que je juge importantes ou intéressantes
